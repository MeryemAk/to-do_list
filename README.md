# To-Do List

A simple to-do list app built with PHP, HTML, and CSS. Users can add and delete tasks.

## Features

- Add tasks
- Delete tasks
- Future updates planned:
  - Pomodoro timer
  - Spotify playlist integration
  - Display current time
